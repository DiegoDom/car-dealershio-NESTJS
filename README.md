## Description

Car dealer ship, my firts NESTJS APP.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run seed data
```bash
# development only
GET: http://localhost:3000/seed
```
